# 🌸 Flor Segura

Aplicação web desenvolvida com **React + Vite** para auxiliar na gestão e monitoramento de floriculturas.

## 🚀 Tecnologias

- React  
- Vite  
- TailwindCSS  
- React Router  
- GitHub Pages (ou outro serviço de deploy)

## ⚙️ Como rodar o projeto localmente

```bash
npm install
npm run dev
